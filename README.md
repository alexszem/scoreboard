# Scoreboard

Code for the scoreboard for the SCC Challengers Berlin. The client for controlling the scoreboard is uploaded @ _tbd_.

## Hardware

- ESP32 Wroom Dev Kit
- P10 Hub 12 1/4 Scan LED Matrices

## Software

- Arduino IDE
- ESP32 board manager by espressif 
- bun
- React
